Terrain:
地形データの製作作成には環境省生物多様性センターが公開する植生図GISデータを使用しています。  
植生図GISデータ：1/25,000植生図（愛知県、岐阜県、福井県、三重県、滋賀県、奈良県）
植生図GISデータ：1/50,000植生図（静岡県、長野県）
For the production of terrain data, we use the vegetation map GIS data released by the Ministry of the Environment Biodiversity Center.
Vegetation map GIS data: 1 / 25,000 vegetation map (Aichi, Gifu, Fukui, Mie, Shiga, Nara)
Vegetation map GIS data: 1 / 50,000 vegetation map (Shizuoka, Nagano)
http://gis.biodic.go.jp/webgis
